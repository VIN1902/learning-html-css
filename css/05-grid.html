CSS Grid

About Grid

Grid is a tool to control layout in a webpage. Usually said to handle 2D layout as compared to 1D layouts of flexbox.

Its a completly different display type than block or inline or flex.

Imagine a checkbox/excel pattern where each number represents the line number.
Each Square is a cell.
Collection of these cells forms rows and columns.

Grid also follows a container-item or parent-child relationship for its properties.
