CSS Flexbox

About Flexbox

Flebox is a tool to control layout in a webpage.

Its a completly different display type than block or inline.

display: flex; creates 2 invisible axis: Main axis and Cross axis.

Main axis is horizontal left-to-right and Cross axis is vertical top-to-bottom.

It also positions all the elements on the main axis at the start.

Using flex-direction property we can switch the axis with one another. Main axis becomes vertical from top-to-bottom and vica-vera.

Flexbox properties are distinguished with parent-child or container-item relationship.
