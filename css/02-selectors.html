CSS Selectors Reference

List of Selectors with Descriptions

Tip: read selectors right-to-left.
  1. Universal Selector (*) = Selects all elements on the page.
  2. Type Selector (element) = Selects all elements of a given type (e.g., p selects all paragraphs).
  3. Class Selector (.class) = Selects all elements with the specified class attribute.
  4. ID Selector (#id) = Selects a single element with the given unique ID.
  5. Attribute Selector (element[attribute]) = Selects elements that have a specific attribute (and optionally a value).
  6. Descendant Selector (ancestor descendant) = Selects elements that are inside (at any depth) another element.
  7. Child Selector (parent > child) = Selects elements that are a direct child of another element.
  8. Adjacent Sibling Selector (A + B) = Selects an element that is immediately after another element (same parent).
  9. General Sibling Selector (A ~ B) = Selects all siblings after a given element (same parent, not necessarily immediate).
  10. Pseudo-class Selector (:hover, :first-child, etc.) = Selects elements based on state/behaviour or position.
  11. Pseudo-element Selector (::before, ::after) = Selects and styles parts of elements (like inserting content before or after).
  12. Grouping Selector (selector1, selector2) = Applies the same styles to multiple selectors at once.
