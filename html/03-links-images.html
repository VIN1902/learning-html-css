Learning about Links and Images

Using Links

Go to Wikipedia

href attribute means hyperlink reference, basically the url.

target attribute is '_blank' for following the link in new tab. '_self' for following the link in same tab (replaces this webpage in the tab). and rest aren't used much.

In rel attribute:
1. noopener: addresses a security vulnerability that arises when you open links in new tabs or windows using the target=”_blank” attribute. A malicious website opened this way could gain partial control of the original window (the one where you clicked the link). The noopener attribute prevents this by severing the connection between the newly opened tab and the original page.
2. noreferrer: is about privacy. When you click a link, the website you land on usually gets information about where you came from (the referrer). The noreferrer attribute hides this referrer information. This can be helpful when linking to external sites that you might not necessarily endorse and want to protect your visitor's privacy.

Using Images

1. use img tag in isolation

2. use figure and figcaption tags for semantic purposes and grouping.

3. use picture and source tags for responsive images (different sizes/formats depending on screen/device). Browser picks the first matching source based on media queries or format support, otherwise falls back to img tag as default.

You can even add image map to images, which allows you to control different actions based on certain parts of an image, like click-events.
